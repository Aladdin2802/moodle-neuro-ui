# Поля базы данных "Методические пособия"

Необходимо создать 4 поля в следующем порядке:

1. **Раздел**
   - **Тип поля:** Переключатель (Radio buttons) или Выпадающий список
   - **Имя поля:** Раздел
   - **Варианты (скопировать как есть):**
     ЧЕРЕПНО-МОЗГОВАЯ ТРАВМА И НЕОТЛОЖНЫЕ СОСТОЯНИЯ
     СОСУДИСТАЯ НЕЙРОХИРУРГИЯ
     НЕЙРООНКОЛОГИЯ
     ФУНКЦИОНАЛЬНАЯ НЕЙРОХИРУРГИЯ
     ОБЩАЯ НЕЙРОХИРУРГИЯ
     ДЕТСКАЯ НЕЙРОХИРУРГИЯ
     ПЕРИФЕРИЧЕСКАЯ И СПИНАЛЬНАЯ НЕЙРОХИРУРГИЯ
     ГНОЙНО-ВОСПАЛИТЕЛЬНЫЕ ЗАБОЛЕВАНИЯ ЦНС

2. **Название**
   - **Тип поля:** Текстовая область (Text input) или Короткий текст
   - **Имя поля:** Название

3. **Файл**
   - **Тип поля:** Файл (File)
   - **Имя поля:** Файл
   - *(Сюда будет загружаться сам файл пособия PDF)*

4. **Ссылка**
   - **Тип поля:** URL-адрес (URL)
   - **Имя поля:** Ссылка
   - *(Для пособий, размещенных на Google Диск или Яндекс Диск)*
