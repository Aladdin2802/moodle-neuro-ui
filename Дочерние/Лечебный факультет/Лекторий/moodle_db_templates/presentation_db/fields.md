# Поля для Базы данных "Презентации кафедры"

В настройках базы данных Moodle (вкладка "Поля") вам необходимо создать 4 поля. Имена полей должны точно совпадать с теми, что указаны ниже (с учетом регистра).

1. **Раздел**
   - **Тип поля:** Меню (Menu)
   - **Имя поля:** Раздел
   - **Опции (впишите ваши разделы столбиком):**
     ЧЕРЕПНО-МОЗГОВАЯ ТРАВМА И НЕОТЛОЖНЫЕ СОСТОЯНИЯ
     СОСУДИСТАЯ НЕЙРОХИРУРГИЯ
     НЕЙРООНКОЛОГИЯ
     ФУНКЦИОНАЛЬНАЯ НЕЙРОХИРУРГИЯ
     ОБЩИЕ ВОПРОСЫ И ПАТОФИЗИОЛОГИЯ
     ФУНКЦИОНАЛЬНАЯ ДИАГНОСТИКА
     ЭНДОСКОПИЧЕСКАЯ НЕЙРОХИРУРГИЯ

2. **Название**
   - **Тип поля:** Текстовая область (Text input) или Короткий текст
   - **Имя поля:** Название

3. **Автор**
   - **Тип поля:** Текстовая область (Text input)
   - **Имя поля:** Автор

4. **Файл**
   - **Тип поля:** Файл (File)
   - **Имя поля:** Файл
   - *(Сюда будет загружаться сама презентация)*

5. **Ссылка**
   - **Тип поля:** URL (URL)
   - **Имя поля:** Ссылка
   - *(Для презентаций больше 320 МБ загружайте на Google Drive и вставляйте ссылку сюда)*
