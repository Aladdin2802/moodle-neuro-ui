# Поля для Базы данных "Экспресс-обзоры"

В настройках базы данных Moodle (вкладка "Поля") вам необходимо создать 4 поля. Имена полей должны точно совпадать с теми, что указаны ниже (с учетом регистра).

1. **Раздел**
   - **Тип поля:** Меню (Menu)
   - **Имя поля:** Раздел
   - **Опции (впишите ваши разделы столбиком):**
     Черепно-мозговая травма
     Нейроонкология
     Сосудистая нейрохирургия
     Спинальная нейрохирургия
     Функциональная нейрохирургия
     Детская нейрохирургия
     Периферическая нервная система
     Гнойно-воспалительные заболевания

2. **Название**
   - **Тип поля:** Текстовая область (Text input) или Короткий текст
   - **Имя поля:** Название

3. **Описание**
   - **Тип поля:** Многострочный текст (Textarea)
   - **Имя поля:** Описание

4. **Автор**
   - **Тип поля:** Текстовая область (Text input)
   - **Имя поля:** Автор

4. **YouTube**
   - **Тип поля:** Текстовая область (Text input)
   - **Имя поля:** YouTube
   - *(Сюда можно вставлять полную ссылку на видео, например: `https://www.youtube.com/watch?v=TPDnsEV-ZAc`)*

5. **RuTube**
   - **Тип поля:** Текстовая область (Text input)
   - **Имя поля:** RuTube
   - *(Сюда можно вставлять полную ссылку на видео)*
